## 3. サービス設計
### 3.1 利用サービスの選定要件
当環境で使用するサービスの選定は以下の要件に基づくものとする。
+ 高いコストパフォーマンスを有していること
+ 簡易な構築、及び運用を可能にすること
+ 高い拡張性を有していること
+ 高いセキュリティを有していること

### 3.2 利用サービス一覧
No.|サービス名|用途|備考
--|--|--|--
1|アマゾン ウェブ サービス (AWS)|IaaS|
